Sven Co-op 4.5 - Global Model Replacement File Instructions
Last updated 12th Match 2009, Adam "AdamR" Reece.


Purpose:

   Global model replacements allows you to replace a model with something else
   on a global basis. This is useful for well themed maps to easily use
   alternative models for weapons, without having to define the model for each
   weapon/ammo entity you put in your maps.

   It can also be used to strip out default weapon models for maps that have the
   "512 model limit exceeded" crash upon loading. You can take out weapon models
   that aren't used in the map to free up some model precache slots.


Instructions:

1: Create a gmr file in the "maps" directory matching a map name (eg. cave.gmr).

2: Copy and paste groups of weapon models from below into the file and save it.

3: Open or create the map cfg (eg. cave.cfg) in the "maps" directory and add
   this line to the end for your map:

   globalmodellist ../../maps/cave.gmr

   "../../" is important, as the working directory doesn't start at "svencoop".

4: That should be it, try and run the map.


THERE IS CURRENTLY NO METHOD OF COMMENTING IN GLOBAL MODEL REPLACEMENT FILES SO
DO NOT COPY COMMENTS FROM BELOW.

DO NOT COPY THE WEAPON/ITEM NAME ON IT'S OWN EITHER. JUST THE REPLACEMENT LINES.

Finally, be aware that you should not take out remove donor specific models just
because you don't expect any donors to be joining your server.
