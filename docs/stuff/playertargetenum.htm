PlayerTargeters

Exists in global namespace.

Values

Name Description
TARGETS_NOALL Disables @all, @admins, @noadmins, @alive, @dead
TARGETS_NOME Disables @me, user can still target self thru steamID or nickname
TARGETS_NOALIVE Disables @alive, also won't return alive players in targeting
TARGETS_NODEAD Disables @dead, also won't return dead players in targeting
TARGETS_NOAIM Disables @aim
TARGETS_NORANDOM Disables @random
TARGETS_NOLAST Disables @last
TARGETS_NONICK Disables nickname & nickname with wildcard
TARGETS_NOIMMUNITYCHECK Disables checking for ACCESS_A
