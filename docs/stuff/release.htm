Preparing for release

When you are ready to release your expansion, be sure to ONLY INCLUDE files required by your expansion

This means any resource or file you made has to be included, but do not include AFBase files

We don't want 200 different versions of AFBase laying around so instead you will have to direct your users to download AFbase first in the release thread

also remember to include the lines for AFBaseExpansions.as, meaning: the include line "#include " and the function call line "(name)_Call();"
